# Book System

A simple Java library management system for handling books, authors, and loans.

## Project Structure

- `src/com/library/Main.java`: Entry point of the application.
- `src/com/library/model/`: Contains model classes (`Author`, `Book`, `Loan`).
- `src/com/library/service/Library.java`: Contains the main library service logic.

## Features
- Manage books and authors
- Handle book loans
- Simple command-line interface

## How to Run
1. Compile the project:
   ```sh
   javac src/com/library/Main.java
   ```
2. Run the application:
   ```sh
   java -cp src com.library.Main
   ```

## Requirements
- Java 8 or higher
